"application"
=============

Contains algorithms to perform SLAM.
"own" is a non-3rd-party algorithm.
The directories contain documentation of how to run on specific datasets.


"datasets"
==========

Contains datasets to perform SLAM on.


"tools"
=======

Contains tools to analyze, correct, and refine SLAM results.
All tools contain documentation, either using a ReadMe file,
or help via command-line (using the "--help" argument).
